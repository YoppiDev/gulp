# GULP Сборка

## Структура папок и файлов

```
├── dev/                          # Исходники
│   ├── js                        # Скрипты
│   │   └── main.js               # Основной скрипт
│   │   ├── vendor                # папка для загрузки локальных библиотек
│   ├── sass                      # Стили сайта
│   │   └── main.scss             # Главный файл стилей
├── ├── img	                      # Изображения
│   │   └── svg             		 # SVG для преобразования в спрайт
│   │   └── webp						 # Картинки преобразовываются в wepb 
├── ├── fonts	                   # Шрифты (в продакшене конвертация в woff(2))
├── ├── resources	                # Прочие ресурсы, например видео

```
